This project contains scripts to generate images and HTML for Fort Collins
Creator Hub's membership payments page.

You may find the results on the "payments" page of our website:
http://www.fortcollinscreatorhub.org/

Contents:

button-img-gen/

  Automates GIMP to create image files for each membership level and payment
  option.

payline-form-gen/

  Generates HTML forms that interface with Payline Data's "QuickClick"
  shopping cart service.
